Important Work

September-February 2021 - Covid-Vis Project In tangent with the CovidVis team as a UROP student, created an interactive story in React about the narrative around Covid-based data visualizations and how they developped over social media. Analyzes over 600,000 tweets with neural networks with the backbone of a flexible data dashboard to explore the large dataset dynamically.

February-August 2020 - CityScanner DataVis Dashboard Created with the CityScanner group as apart of a UROP. Made in JavaScript, HTML, CSS, this is a web-based data visualization dashboard integrated with Mapbox and D3-based graph widgets to show air quality data. The site is very flexible and can be easily adjusted via configuration files to deploy for different cities and with different data overlays.

August 2019 - WIRE Project Website A custom website designed with HTML, CSS, and Bootstrap made completely by hand by Anna A-D for the University of Wisconsin, Madison. It is mobile optimized and I generated custom maps for the background images using Python and Geopandas.

January-May 2019 - City of Madison Data Science Project A study of the Madison City Budget using data science in cooperation with Tyler Harter and students at UW Madison. I specifically focused on the Fire department and analyzing location of incidents, incident frequency, and overtime pay. I used Python, Jupyter Notebooks, and Pandas.

June 2020 - Emo(na)tion A React/Node.js/MongoDB website in which users can log-in via Google Auth and place a marker on an emotional landscape representing where they are emotionally. A new form of social network. GitHub Repo.

January 2020 - Pass the Recipe A React/Node.js/MongoDB website created as apart of the MIT WebLab IAP class. It is a recipe website, but unlike a traditional recipe website, once you “Pass the Recipe” anyone can suggest edits, vote to find the best directions, and transform your recipe into something new.

June 2018 - Political Computing. A work in progress website, home to research and opinions about political science and its interaction with CS. Also hosts the JavaScript Gerrymandering Sandbox.

November 2018 - Alliance Online. Made in 24 hours by Anna A-D and Langston N., Alliance Online is a multiplayer strategy board game made in Python, Flask, Flask-SocketIO, JavaScript, and Postgresql. GitHub Repo.

March 2019 - Anna's Corner Made in less than a day to learn about node.js and Postgresql, Anna's Corner is a simple blogging site. Drop by and leave a post! GitHub Repo.

May-June 2018 - Dungeon Cubed. Made over the course of several months by Anna A-D and Langston N., Dungeon Cubed is a isometric/3D dungeon crawling rougelike made in Unity/C# with inventory, enemies, skill/leveling, weapons, and 3 demo levels. GitHub Repo.
